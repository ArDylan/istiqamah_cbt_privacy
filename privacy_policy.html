Aplikasi Istiqamah Exam dikembangkan oleh Tim IstiqamahOne sebagai aplikasi resmi ujian online untuk siswa SD, SMP, dan SMA Istiqamah YPAIT Balikpapan. Kebijakan privasi ini menjelaskan bagaimana aplikasi ini mengelola data pengguna, terutama untuk menjaga keamanan dan kenyamanan siswa selama pelaksanaan ujian.

1. Informasi yang Kami Kumpulkan

Aplikasi tidak mengumpulkan data pribadi pengguna seperti:

  • Nama lengkap
  • Email
  • Nomor telepon
  • Lokasi
  • Foto atau video

Satu-satunya data yang disimpan secara lokal di perangkat adalah:

  • Alamat server ujian (IP Server)
  • Kode ujian (Exam Key)
  • Unit atau jenjang (SD, SMP, SMA)

Data ini hanya digunakan untuk mempercepat proses masuk ke halaman ujian dan tidak dikirim ke server lain selain server sekolah.

2. Keamanan Data

Kami menerapkan langkah-langkah keamanan untuk melindungi data dari akses yang tidak sah. Seluruh komunikasi antara aplikasi dan server sekolah menggunakan protokol HTTP/HTTPS yang hanya digunakan dalam jaringan sekolah atau lingkungan ujian resmi.

Aplikasi ini tidak meminta izin untuk mengakses lokasi, kamera, mikrofon, kontak, atau penyimpanan pribadi pengguna.

3. Penggunaan oleh Anak-Anak

Karena aplikasi ini digunakan oleh siswa dari tingkat Sekolah Dasar hingga Sekolah Menengah Atas, kami berkomitmen mematuhi Google Play Families Policy:

  • Konten aplikasi sepenuhnya aman untuk anak-anak.
  • Aplikasi tidak menampilkan iklan.
  • Tidak ada pelacakan atau analisis perilaku pengguna.
  • Tidak ada akses terhadap data pribadi anak-anak.

4. Koneksi Internet

Aplikasi ini membutuhkan koneksi internet hanya untuk:

  • Mengambil data ujian dari server sekolah.
  • Membuka halaman ujian melalui WebView.

Aplikasi tidak mengirimkan data lain selain yang diperlukan untuk pelaksanaan ujian.

5. Akses Pihak Ketiga

Aplikasi tidak menggunakan layanan pihak ketiga (seperti Google Ads, Firebase Analytics, atau SDK iklan lainnya) yang dapat mengumpulkan data pengguna.
